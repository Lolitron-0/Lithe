Detailed Description

template<class Derived>
class Lithe::CameraController< Derived >

Base class for user-defined camera controllers Camera controllers are wrappers around camera object, handling user input. There are a bunch of methods to optionally override, they should be called in corresponding layer methods.

Attention
OnEvent call is necessary for the event handlers to work
Note
This class uses CRTP to dispatch events. So deriving should look like this
class MyController : public CameraController<MyController>
{
// ...
};
