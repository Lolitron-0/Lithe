Detailed Description

Framebuffer object abstraction.

Attachments can be set up in FramebufferProperties object

props.Attachments = {TextureFormat::RGBA8, TextureFormat::Depth};

virtual RendererId Ra::Framebuffer::GetColorAttachmentHandle ( std::size_t  index = 0) const
pure virtual

Returns color attachment with given index.

Warning
This method does exactly what it's name says, so in case of multisampled framebuffer, multisampled texture will be returned. To receive just the 2d texture to draw use GetDrawTextureHandle();
