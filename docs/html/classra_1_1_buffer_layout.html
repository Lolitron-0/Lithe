Detailed Description

Class describing data arrangement in VertexBuffers.

Layout can be defined as follows:

BufferLayout layout = {
{BufferElementDataType::Float3, "myPosition"},
{BufferElementDataType::Float2, "myTexCoord"},
{BufferElementDataType::Bool, "myBoolForSomethingImportant"},
}
